일단 자를만 한지를 판단해준다.

가로로 자를 때, 세로로 자를 때를 생각해주고
자를지, 다 잘렸는지, 자를수가 없는지(자를 가치가 없는지)를 판단해준다.

만약 해당 판에 보석이 없다? 이건 실패다. return 0
보석이 하나이상은 있어야 하고
보석이 1갠데 불순물이 없다? 이럼 성공(다 잘림) return 1
보석 1갠데 불순물이 1개이상? 이럼 실패 return 0
보석 2개이상인데 불순물 0개? 실패 return 0
불순물이 있다? 자를 가치가 있다!
