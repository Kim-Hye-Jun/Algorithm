이분탐색 문제이다.

답은
1) 알칼리 2개 인경우
2) 산성 2개 인경우
3) 알칼리1개 산성1개 인경우가 있고

1),2)의 경우에는 가장 절댓값이 작은 두 용액이 정답이니 그걸 구해주면되고

3)의 경우에 이분탐색을 쓴다.
알칼리 , 산성 용액 각각 다른 배열에 받고
알칼리 용액마다 그에 맞는(절댓값을 줄여줄) 산성용액을
이분탐색으로 찾는다( 절댓값이 작거나 같은 용액 중 가장 큰용액)
그리고 그 용액과 그것보다 하나 큰 용액을(있다면) 찾고
비교한다.
모든 알칼리 용액을 반복문으로 돌리며 찾고
마지막에 있는값을 출력한다.

##

int BinarySearch(int value)
{
  int mid;
  int start = 0;
  int end = P.size()-1;

  while(start+1<end)
  {
    mid = (start+end)/2;
    if(P[mid]>value)
      end = mid;
    else
      start = mid;
  }

  return start;
}
